## Sobre o Projeto

Este projeto é um relógio analógico construído com foco em simplicidade, clareza e boa experiencia visual.

Com estrutura enxuta e sem frameworks, ele demonstra um fluxo completo de atualização de interface em tempo real no front-end:

- Atualização dos ponteiros de segundos, minutos e horas com base no horário atual
- Interface limpa e objetiva
- Separação clara entre estrutura, estilo e lógica
- Layout adaptado para desktop e dispositivos móveis

## Destaques do Projeto

Mesmo sendo um projeto pequeno, entrega valor prático e fácil manutenção:

- Zero complexidade de framework
- Código leve e fácil de entender
- Atualização do relógio em tempo real com JavaScript
- Estilização responsiva em arquivo dedicado para mobile
- Estrutura organizada para evolução futura

## Tecnologias Utilizadas

- **HTML5** - estrutura semântica da interface do relógio
- **CSS3** - estilos visuais, posicionamento e acabamento
- **JavaScript (Vanilla)** - lógica de tempo e rotação dos ponteiros
- **Google Fonts (Poppins)** - tipografia da interface
- **Media Queries** - ajustes para diferentes tamanhos de tela

## Funcionalidades

- Exibição de relógio analógico em tempo real
- Ponteiro de segundos com atualização a cada segundo
- Ponteiros de minutos e horas sincronizados com o horário atual
- Interface responsiva para mobile
